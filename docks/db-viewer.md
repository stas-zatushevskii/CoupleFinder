# DB Viewer

`db-viewer` — это отдельный маленький сервис для удобного просмотра содержимого базы данных через браузер.

## Что делает

Сервис:

- показывает список пользователей из базы
- отображает анкетные данные
- показывает предпочтения пользователя
- выводит интересы
- выводит вредные привычки
- позволяет искать по имени или городу
- позволяет фильтровать по полу

## Где находится

Обычно структура выглядит так:

```text
db-viewer/
├── package.json
├── server.js
└── public/
    ├── index.html
    ├── styles.css
    └── app.js
```

## Как запускать локально

Перейти в папку проекта:

```bash
cd db-viewer
```

Установить зависимости:

```bash
npm install
```

Запустить сервис:

```bash
DATABASE_URL="postgres://postgres:postgres@localhost:5433/couplefinder" npm start
```

После запуска страница будет доступна по адресу:

```text
http://localhost:4000
```

## Переменные окружения

### `DATABASE_URL`
Строка подключения к PostgreSQL.

Пример:

```text
postgres://postgres:postgres@localhost:5433/couplefinder
```

### `PORT`
Порт, на котором будет слушать сервис.

По умолчанию используется:

```text
4000
```

Пример запуска с указанием порта:

```bash
PORT=4000 DATABASE_URL="postgres://postgres:postgres@localhost:5433/couplefinder" npm start
```

## API

Сервис поднимает внутренний endpoint:

```text
GET /api/users
```

Он используется фронтендом viewer-страницы для загрузки данных из базы.

### Поддерживаемые query-параметры

#### `search`
Поиск по имени или городу.

Пример:

```text
/api/users?search=chi
```

#### `gender`
Фильтр по полу.

Пример:

```text
/api/users?gender=female
```

## Когда использовать

`db-viewer` удобен для:

- ручной проверки, что генератор действительно записал данные
- просмотра текущего состояния базы
- проверки интересов, предпочтений и вредных привычек пользователей
- визуальной отладки перед запуском алгоритмов
